# Verified result artifacts

These files are committed evidence, not marketing benchmarks. Every number is scoped to the
included synthetic fixture or smoke input.

## Contract and system results

- `cpu_bronze.json`: deterministic 120-question EAMM-Bench Bronze contract run.
- `ablations.json`: full/no-graph/top-1/no-visual-gate comparison on the same fixture.
- `api_load_local.json`: local Windows API concurrency smoke.
- `api_load_autodl.json`: AutoDL API concurrency smoke using the deterministic CPU path.

Latency and throughput are machine-specific and exclude ASR, OCR, embedding-model, and LLM
inference unless an artifact explicitly says otherwise.

## RTX 4090 integration results

The `gpu/` directory contains real-model integration reports from one AutoDL RTX 4090:

- `environment.json`: driver, CUDA, package versions, and exact cached model commits;
- `ocr_environment.json`: Paddle/PaddleOCR/PaddleX versions, CUDA runtime packages, and model-file hashes;
- `asr_small_4090.json`: faster-whisper transcript, timestamps, WER, and runtime;
- `bge_m3_4090.json`: multilingual retrieval scores and runtime;
- `qwen3_8b_4090.json`: evidence-constrained answer, citation/fact gates, runtime, and peak VRAM;
- `ocr_4090.json`: PaddleOCR atoms extracted from the generated slides;
- `diarization_fallback_smoke.json`: license-free turn-detection Plan B and its explicit limits;
- `demo_media_sha256.txt`: hashes for the CC0 release media generated by the repository script.

Missing files mean that integration gate has not been verified. The synthetic demo media is
published as a release asset rather than committed to Git.
